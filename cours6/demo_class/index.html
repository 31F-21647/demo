Les classes et les objets en javaScript

Informations

Créer un formulaire d'inscription sur 3 pages (info personnelles, inscriptions, horaire)

4 classes :

  • App : Gère l'application, l'affichage générale et les appels sur les divers "composants"
  • Information : Composant qui gère la section "Informations personnelles"
  • Inscription : Composant pour l'inscription aux cours
  • Horaire : Composant pour le choix de l'horaire

À venir : des routes, un moteur de gabarit, des données dynamiques provenant du serveur...
